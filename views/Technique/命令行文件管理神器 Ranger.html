      命令行文件管理神器 Ranger

      Kyrin 2020-03-10 linux mac utility

      介绍 Unix 类系统的命令行文件管理器 Ranger 的基本用法和高阶配置

      # Ranger

      这段时间突然对自己的帅气程度有了更高的要求，实现渠道之一就是尽量所有的事情都在终端完成，在他人都看不懂的情况在一行行代码中完成任务，于是对于 GUI 深恶痛绝，找各种软件的终端替代品。Ranger 就是我自我升华过程中的一个发现，为此我也花了不少功夫去学习和定制它。

      Ranger 的功能很全，Vim like 的控制命令、多列文件管理方式、支持多标签页、实时预览等深深吸引着我。

      # 预览视频缩略图

      取消 ~/.config/ranger/scope.sh 里的 handle_image 方法里 video 的注释。

      安装所需要的 ffmpegthumbnailer（mac 上用 brew 安装）。

      # 预览pdf

      推荐用pdf转图片的方式预览，同样在 scope.sh 里的 handle_image 方法里取消 pdf 的注释。

      安装所需的 pdftoppm，它集成在 pdf 处理软件 poppler 当中（mac 上用 brew 安装，需要依赖 qt）。

      # 色彩设置

      基于 ~/.config/ranger/colorschemes 下的文件修改，修改后记得将 rc.conf 中 set colorscheme 设为自己的配置文件。

      其实还可以预览 doc、docx、xls、xlsx、md 等文件类型，均在 scope.sh 文件中有提示加入支持，结合 Pandoc 可以预览。

      # 插件安装

      参考 https://github.com/ranger/ranger/wiki/Plugins

      ranger_devicons

      依赖 nerd font，令 ranger 更加赏心悦目，推荐本人喜欢的 Monaco Nerd Font 字体。

      ranger-autojump

      前提是需要已安装 autojump，根据 READ.ME 将 autojump.py 拷贝至 ~/.config/ranger/plugins/ 内。在 ranger 中输入：

      :j <dir>
      1

      即可在 ranger 中自动跳转目录。

      该项目还提供了方便的在 zsh 中直接用 autojump 打开 ranger 的脚本，将 ranger-autojump.plugin.zsh 中的内容复制进 ~/.zshrc 中即可。在 zsh 中输入：

      r #打开 ranger
r <dir> #通过 autojump 在 ranger 中打开目录

      # 定义函数

      效果：通过大写 Q 退出后 shell 进入 ranger 最后所处的目录（兼容 bash 的 shell，fish 不行）

      将函数添加至 ~/.bash_profile 或 ~/.zrhrc

      function ranger {
    local IFS=$'\t\n'
    local tempfile="$(mktemp -t tmp.XXXXXX)"
    local ranger_cmd=(
        command
        ranger
        --cmd="map Q chain shell echo %d > "$tempfile"; quitall"
    )

    ${ranger_cmd[@]} "$@"
    if [[ -f "$tempfile" ]] && [[ "$(cat -- "$tempfile")" != "$(echo -n `pwd`)" ]]; then
        cd -- "$(cat "$tempfile")" || return
    fi
    command rm -f -- "$tempfile" 2>/dev/null
}
