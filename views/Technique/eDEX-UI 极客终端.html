      eDEX-UI 极客终端

      Kyrin 2020-11-29 linux mac utility

      偶然发现的超级炫酷全平台终端，但是网络显示有些问题，因此对源代码做出些小改动，特此记录。

      # 安装

      eDEX-UI Github

      # brew cask 安装

      # 有下面的网络问题的话，还得靠源码安装
brew cask install edex-ui
brew cask uninstall edex-ui

      2
      3

      # 源码安装

      git clone https://github.com/GitSquared/edex-ui.git
cd edex-ui

# start
npm run install-linux
npm start

# build
## 在dist目录中生成打包文件
npm install
npm run build-linux/build-darwin/build-windows

      # 快捷键

      Shift-Ctrl-K 打开快捷键列表
      Shift-Ctrl-S 打开设置

      更改设置后，选择 Save to Disk 将设置写入 ~/Library/Application Support/eDEX-UI/settings.json，再选择 Reload UI。

      # 网络问题修复

      安装成功后，右侧总是无法访问网络，经排查，根源问题在 ./src/classes/netstat.class.js 文件的 ping 函数中的 connect() 无法建立连接。

      • 修改106行：host: "www.myexternalip.com"
      • 140行后增加一行：offline = false

      这样做，仅仅是绕过了建立连接查询时延的过程，使其他功能不受这个问题影响，但 PING 的值会显示为 NaN。
