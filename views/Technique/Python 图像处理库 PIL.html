    介绍 Python 图像处理库 PIL 的用法

    # PIL

    PIL (Python Imaging Library) 是 Python 图像处理库，该库支持多种图像格式，提供强大的图像处理功能。以前自己做分割图像的小工具时有接触过，不过这次是在帮一个同学搞 CT 图像的时候在此好好学习一下，特此记录所学。

    # 读写操作

    # 导入 PIL

    from PIL import Image
    1

    # Image.open 打开图像文件

    img=Image.open('image.jpg')
    1

    # Img.show 显示图像

    img.show()
    1

    # 导入 numpy

    import numpy as np
    1

    # np.array or np.asarray 将图片转化成 numpy 数组

    img_arr = np.array(img)
    1

    # Image.fromarray 将numpy数组转化为图片

    new_img = Image.fromarray(np.unit8(img_arr))
    1

    # img.save 存储转化图片

    可以转换为其他格式存储图片（PNG、BMP 和 JPG 彩色图像格式之间的互相转换都可以通过 Image 模块的 open() 和 save() 函数来完成）

    new_img.save('new_image.png')
    1

    print(img.format) #JPEG

    # img.convert() 不同模式图像之间的转换。

    PIL的九种不同模式：1、L、P、RGB、RGBA、CMYK、YCbCr、I、F：

    Mode Intro
    1 1位像素，表示黑和白，但是存储的时候每个像素存储为 8bit。
    L 8位像素，表示黑和白。从模式 RGB 转换为 L 模式是按照下面的公式转换的：L = R * 299/1000 + G * 587/1000+ B * 114/1000。
    P 8位像素彩色图像，使用调色板映射到其他模式。alpha通为透明通道，转换成 RGB 时设置为255完全不透明
    RGB 3x8位像素，为真彩色。
    RGBA 4x8位像素，有透明通道的真彩色。
    CMYK 4x8位像素，颜色分离。四种标准颜色是：Cyan 青色、Magenta 洋红色、Yellow 黄色、Key Plate(blacK) 定位套版色（黑色）。PIL中 RGB 转换为 CMYK 的公式为：C = 255 - R、M = 255 - G、Y = 255 - B、K = 0。由于该转换公式比较简单，转换后的图像颜色有些失真。
    YCbCr 3x8位像素，彩色视频格式。YCbCr其中Y是指亮度分量，Cb指蓝色色度分量，而Cr指红色色度分量。人的肉眼对视频的Y分量更敏感，因此在通过对色度分量进行子采样来减少色度分量后，肉眼将察觉不到的图像质量的变化。
    I 32位整型灰色图像。其他同模式 L 相同。
    F 32位浮点灰度图像。其他同模式 I 相同。
    print(img.mode) #'RGB'
new_img = img.convert("RGBA")
print(new_img.mode) #'RGBA'

    # img.transpose 图像翻转或旋转

    img.transpose(method)
    1

    method 是 transpose 的参数：

    Method Intro
    Image.FLIP_LEFT_RIGHT 图像左右翻转
    Image.FLIP_TOP_BOTTOM 图像上下翻转
    Image.ROTATE_90 图像逆时针旋转90°
    Image.ROTATE_180 图像逆时针旋转180°
    Image.ROTATE_270 图像逆时针旋转270°
    Image.TRANSPOSE 图像转置
    Image.TRANSVERSE 图像转置再水平翻转

    # ImageEnhance 图像增强

    PIL的 ImageEnhance 类专门用于图像增强处理，可以增强（减弱）图像的亮度 Brightness、对比度 Contrast、色度 Color、以及锐度 Sharpness。

    from PIL import Image
from PIL import ImageEnhance
 
# 原始图像
img = Image.open('image.jpg')
# 亮度增强
img_bright = ImageEnhance.Brightness(imag).enhance(3)
image_bright.show()
# 色度增强
img_color = ImageEnhance.Color(img).enhance(2)
image_color.show()
# 对比度增强
img_contrast = ImageEnhance.Contrast(img).enhance(3)
image_contrast.show()
# 锐度增强
img_sharp = ImageEnhance.Sharpness(img).enhance(2)
image_sharp.show()

    # ImageSequence 处理图像序列

    # 遍历 gif 图像中的所有帧，并分别保存为图像
from PIL import Image
from PIL import ImageSequence

gif = Image.open("cxk.gif")
for i,frame in enumerate(ImageSequence.Iterator(gif),1):
    if frame.mode == 'JPEG':
        frame.save("%d.jpg" %i)
    else:
        frame.save("%d.png" %i)

    # 除了上面使用迭代器的方式以外，还可以一帧一帧读取 gif，读取到 gif 的最后一帧之后，会throw 一个 EOFError，只要捕获这个异常就可以
index = 0
while 1:
    try:
        gif.seek(index)
        gif.save("%d.%s" %(index,'jpg' if gif.mode == 'JPEG' else 'png'))
        index += 1
    except EOFError:
        print("Reach the end of gif sequence!")
