    Mac 配置 zsh

    Kyrin 2020-03-12 mac zsh

    # 切换 shell

    苹果官网给出了更改默认 shell 的建议

    bash 是版本较早的 shell，zsh 基本兼容 bash 且功能更强大。

    自升级到 macOS Mojave 后，发现打开终端会弹出一条警告，提示默认交互 shell 为 zsh。若仅想消除这个警告，可以将以下命令添加到 ~/.bash_profile中。

    export BASH_SILENCE_DEPRECATION_WARNING=1
    1

    切换 zsh 作为 shell 则在终端输入下行命令，输入密码重新打开终端生效：

    chsh -s /bin/zsh
    1

    同理，切换回 bash：

    chsh -s /bin/bash
    1

    将提示符配置、描述文件及环境信息从 .bash_profile 和 .bashrc 中转移至 .zshrc 文件中。

    其中提示符的配置在zsh中已不太一样：

    # Prompt color
autoload -U colors && colors #开启颜色
PROMPT="%{$fg[yellow]%}%n%{$reset_color%}@%{$fg[green]%}%m %{$fg[blue]%}%1|%~ %{$reset_color%}%#> " #配置提示符
RPROMPT="[%{$fg_bold[yellow]%}%?%{$reset_color%}]" #在行末显示上一命令的返回状态

    到目前为止，已经将 shell 迁移成功。

    # Oh-My-Zsh 主题

    # 安裝 oh-my-zsh

    sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
    1

    oh-my-zsh 会将原来的 .zshrc 文件备份到 .zshrc.pre-oh-my-zsh 文件。先来配置 zsh 皮肤。

    # 安装 powerlevel10k

    git clone https://github.com/romkatv/powerlevel10k.git $ZSH_CUSTOM/themes/powerlevel10k
    1

    # 配置 powerlevel10k

    修改 zsh 的主题为 powerlevel10k

    ZSH_THEME="powerlevel10k/powerlevel10k"
    1

    重启终端或者输入：

    source ./zshrc
    1

    进入powerlevel10k主题的配置问答界面，每一次想要重新配置，可以在终端输入命令：

    p10k configure
    1

    配置完毕后会生成 ~/.p10k.zsh 文件，编辑该文件亦可更改配置。

    # 添加 oh-my-zsh 插件

    zsh-autosuggestions 命令自动补全插件

    git clone https://github.com/zsh-users/zsh-autosuggestions.git $ZSH_CUSTOM/plugins/zsh-autosuggestions
    1

    zsh-syntax-highlighting 终端命令语法高亮

    git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM}/plugins/zsh-syntax-highlighting
    1

    安装 autojump

    brew install autojump
    1

    # 配置插件

    在~/.zshrc文件找到 plugins=(git) 修改

    plugins=(
  git
  autojump
  zsh-autosuggestions
  zsh-syntax-highlighting
)

    安装 zsh-autosuggestions 插件后会出现粘贴内容慢的情况，在项目的issues#238 中有讨论，将如下代码加入到 ~/.zshrc 中。

    # This speeds up pasting w/ autosuggest
# https://github.com/zsh-users/zsh-autosuggestions/issues/238
pasteinit() {
  OLD_SELF_INSERT=${${(s.:.)widgets[self-insert]}[2,3]}
  zle -N self-insert url-quote-magic # I wonder if you'd need `.url-quote-magic`?
}

pastefinish() {
  zle -N self-insert $OLD_SELF_INSERT
}
zstyle :bracketed-paste-magic paste-init pasteinit
zstyle :bracketed-paste-magic paste-finish pastefinish

    zsh-syntax-highlighting 默认的合法命令是绿色，在终端如果比较晃眼可以在 ~/.zshrc 文件添加：

    ZSH_HIGHLIGHT_STYLES[arg0]=fg=15 #fg=color 是按照xterm-256color设置，15是白色
    1

    xterm-256color 可以对照该表查看。

    # 还原环境变量

    将 .zshrc.pre-oh-my-zsh 文件中的内容复制回 .zshrc 文件。

    刷新环境变量：

    source ~/.zshrc
