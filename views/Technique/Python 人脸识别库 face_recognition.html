    Python 人脸识别库 face_recognition

    Kyrin 2020-03-28 python cv

    介绍很好用的 Python 人脸识别库 face_recognition

    在做一个好玩的 颜值检测网站 时用到了这个库，在此做一下记录。

    # face_recognition

    官方说明：https://face-recognition.readthedocs.io/en/latest/index.html

    face_recognition 用 dlib 最新的基于深度学习的人脸识别，是最简单的人脸识别库，可以通过 Python 和 命令行识别和处理人脸。模型在 Labeled Faces in the Wild 基准有99.38%的准确率。

    # 安装

    pip install face_recognition
    1

    # API

    目前只用到过两个方法，但库远比这能做的多，具体的函数及详细参数等请查阅官网。

    import face_recognition as fr

# Load image
# Returns: image contents as numpy array
image = fr.load_image_file("img.jpg")

# Find all the faces in the image
# Returns: A list of tuples of found face locations in css (top, right, bottom, left) order
face_locations = fr.face_locations(image)

# Find the facial features in the image
# Returns: A list of dicts of face feature locations (eyes, nose, etc)
face_landmarks_list = fr.face_landmarks(image)

# Get face encodings for each face in the image:
# Returns: A list of 128-dimensional face encodings (one for each face in the image)
list_of_face_encodings = fr.face_encodings(image)

# Compare a list of face encodings against a candidate encoding to see if they match
# Returns: A list of True/False values indicating which known_face_encodings match the face encoding to check
results = fr.compare_faces(known_face_encodings, a_single_unknown_face_encoding)
