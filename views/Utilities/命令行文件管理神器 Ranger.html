        # 操作命令

        ? #查看帮助

hjkl←↓↑→ # 浏览文件

I #重命名，光标在当前文件名前
yy #复制
dd #剪切

pl #创建一个被复制或剪切文件的符号链接
pL #创建一个被复制或剪切文件的符号链接（相对路径）

w #打开或关闭任务管理视图
W #打开Message Log

J #降低当前任务的优先级
K #提升当前任务的优先级
M #选择显示模式

:delete #删除选定条目
:delete <filename> #删除文件
:chmod <octal> #设置被选条目权限
:filter <string> #只显示匹配字符串的文件

:mark <regexp> #选定匹配基本正则表达式的所有文件
:unmark <regexp> #取消选定匹配基本正则表达式的所有文件
:mkdir #创建目录
:touch <filename> #创建文件

z #修改设置，只对当前进程生效，下一次打开ranger的配置还是去rc.conf读取
zp #开关文件预览，可以默认不预览(在rc.conf中修改set preview_files false)，想预览时zp，加快浏览速度
zh | Backspace #开关显示隐藏文件
u #撤销操作
R #刷新当前目录
Ctrl-r #清空缓存并刷新目录
Ctrl-l #重画当前窗口

        # 预览视频缩略图

        取消~/.config/ranger/scope.sh里的handle_image方法里video的注释

        安装所需要的ffmpegthumbnailer（mac上用brew安装）

        # 预览pdf

        推荐用pdf转图片的方式预览，同样在scope.sh里的handle_image方法里取消pdf的注释

        安装所需的pdftoppm，它集成在pdf处理软件poppler当中（mac上用brew安装，需要依赖qt）

        # 色彩设置

        基于~/.config/ranger/colorschemes下的文件修改，修改后记得将rc.conf中set colorscheme设为自己的配置文件

        # 其他设置

        其实还可以预览doc、docx、xls、xlsx、md等文件类型，均在scope.sh文件中有提示加入支持，本人对其需求较低便不在此赘述

        # 插件安装

        https://github.com/ranger/ranger/wiki/Plugins

        devicons插件，依赖nerd font，令ranger更加赏心悦目，推荐本人喜欢的Monaco Nerd Font字体

        # 定义函数

        效果：通过大写Q退出后shell进入ranger最后所处的目录（兼容bash的shell，fish不行）

        将函数添加至~/.bash_profile

        function ranger {
    local IFS=$'\t\n'
    local tempfile="$(mktemp -t tmp.XXXXXX)"
    local ranger_cmd=(
        command
        ranger
        --cmd="map Q chain shell echo %d > "$tempfile"; quitall"
    )

    ${ranger_cmd[@]} "$@"
    if [[ -f "$tempfile" ]] && [[ "$(cat -- "$tempfile")" != "$(echo -n `pwd`)" ]]; then
        cd -- "$(cat "$tempfile")" || return
    fi
    command rm -f -- "$tempfile" 2>/dev/null
}
